# Express JWT Authentication API

A simple authentication system built with **Node.js**, **Express**, **MongoDB**, **bcrypt**, and **JWT**.  
Users can register, login, and access a protected profile route using token-based authentication.

---

## Features

- User Registration with hashed password
- User Login with JWT token generation
- Protected route using JWT Middleware
- Secure password handling using bcrypt
- MongoDB database connection
- Token verification and user authorization

---

## Technologies Used

- **Node.js**
- **Express.js**
- **MongoDB**
- **bcrypt**
- **jsonwebtoken**
- **dotenv**
